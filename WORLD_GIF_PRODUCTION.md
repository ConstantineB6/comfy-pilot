## TECHNICAL SPECIFICATIONS

### Video Specs
```
Resolution: 1080p (1920x1080) or 1440p (2560x1440)
Frame Rate: 30fps (plays smoothly on all platforms)
Duration: 60 seconds
Codec: H.264 (MP4) or VP9 (WebM)
File Size Target: 5-15 MB
Aspect Ratio: 16:9 (standard for web)
```

### Audio Specs
```
Sample Rate: 48 kHz
Bit Depth: 16-bit
Channels: Stereo
Volume: -6dB to -3dB (leaves headroom)
Music: 40-50% of mix
Sound Effects: 20-30% of mix
Voice: 30-40% of mix (if used)
```

### Color Grading

**Before (Left Side):**
```
Temperature: Cool (5500K)
Saturation: -30% (desaturated, muted)
Contrast: High (shadows deeper, highlights blown)
Shadows: Blue-ish tint
Highlights: Harsh screen light
Overall: Exhaustion, urgency, darkness
```

**After (Right Side):**
```
Temperature: Warm (7000K, sunrise simulation)
Saturation: +20% (vibrant, alive)
Contrast: Softer (shadows still visible, highlights preserved)
Shadows: Natural depth
Highlights: Soft and forgiving
Overall: Energy, clarity, hope
```

**Threshold (Lightning Crack):**
```
Brightness: +30% flash
Desaturation: Brief moment
Then transition to warm/saturated
Creates visual shock and transformation
```
